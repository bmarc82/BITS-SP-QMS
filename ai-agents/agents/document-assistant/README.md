# Dokumenten-Assistent

## Zweck
Hilft Process Ownern beim Erstellen neuer Prozessbeschreibungen im vorgesehenen Format.

## Eingabe
Freitext: «Beschreibe den Prozess in einfachen Worten»

## Ausgabe
Strukturiertes Prozessdokument:
- Zweck und Geltungsbereich
- Prozessschritte (nummeriert)
- Verantwortlichkeiten (RACI)
- Verweise auf ISO-Kapitel
- Messgrößen und KPIs

## Wissensquelle
- QMS-Dokumentenbibliothek (freigegebene Dokumente)
- ISO 9001:2015 Normauszüge (hochgeladen)
- Unternehmensspezifische Vorlagen
