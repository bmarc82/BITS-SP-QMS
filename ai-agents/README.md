# KI-Agenten Integration

Intelligente Unterstützung bei Erstellung, Pflege und Verbesserung von QMS-Dokumenten.

## Agenten

| Agent | Beschreibung |
|-------|-------------|
| document-assistant | Hilft beim Erstellen neuer Prozessbeschreibungen |
| review-advisor | Analysiert Dokumente vor dem Review |
| audit-prep-agent | Bereitet Auditunterlagen vor |
| process-optimizer | Schlägt Prozessverbesserungen vor (KVP) |

## Technologie

- Microsoft Copilot Studio (primär)
- Azure AI Foundry (erweiterte Agenten)
- SharePoint als Wissensbasis

## Datenschutz

Alle Agenten greifen nur auf **freigegebene** Dokumente zu.  
Entwürfe und vertrauliche Inhalte sind ausgeschlossen.
