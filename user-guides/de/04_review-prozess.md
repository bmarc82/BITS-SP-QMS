# Review und Aktualisierung

## Review-Erinnerungen

Sie erhalten automatische Erinnerungen:
- **60 Tage vor Fälligkeit**: Erste Erinnerung
- **30 Tage vor Fälligkeit**: Zweite Erinnerung
- **7 Tage vor Fälligkeit**: Dringende Erinnerung

## Review durchführen

1. Öffnen Sie das Dokument über den Link in der Erinnerung
2. Prüfen Sie den Inhalt auf Aktualität
3. Wählen Sie eine der Optionen:
   - **Keine Änderung notwendig**: Review bestätigen ohne Änderungen
   - **Aktualisierung erforderlich**: Dokument bearbeiten und neu einreichen
4. Bestätigen Sie das Review

## Nächstes Review-Datum

Das nächste Review-Datum wird automatisch basierend auf der konfigurierten Gültigkeitsdauer gesetzt.
