# KI-Agenten nutzen

## Dokumenten-Assistent

Für die Erstellung neuer Prozessdokumente:

1. Öffnen Sie den KI-Assistenten im QMS-Portal oder in SharePoint
2. Beschreiben Sie den Prozess in einfachen Worten
3. Wählen Sie die ISO-Kapitel (optional)
4. Klicken Sie auf «Dokument erstellen»
5. Überprüfen und bearbeiten Sie den generierten Entwurf

## Review-Berater

Vor einem Review:

1. Öffnen Sie das Dokument
2. Klicken Sie auf «KI-Analyse starten»
3. Prüfen Sie die Analyse auf Lücken und Verbesserungsvorschläge
4. Entscheiden Sie, welche Vorschläge Sie übernehmen möchten

## Datenschutz

Der KI-Assistent greift nur auf **freigegebene** QMS-Dokumente zu. Entwürfe und vertrauliche Dokumente sind ausgeschlossen.
