# Admin Tool – Benutzerhandbuch

## Verbindung konfigurieren

1. Starten Sie das Admin Tool
2. Öffnen Sie «Einstellungen» > «Verbindung»
3. Geben Sie ein:
   - Tenant-URL (z.B. `https://contoso.sharepoint.com`)
   - Application (Client) ID
   - Zertifikat-Thumbprint oder Client Secret
   - SharePoint Site-URL
4. Klicken Sie auf «Verbindung testen»
5. Bei Erfolg: «Speichern»

## Module installieren

1. Öffnen Sie «Installation» > «Module»
2. Wählen Sie die zu installierenden Module aus
3. Prüfen Sie die Abhängigkeiten (automatisch)
4. Klicken Sie auf «Installieren»
5. Verfolgen Sie den Fortschritt im Protokoll

## Wartung

- **Backup**: «Wartung» > «Konfiguration exportieren»
- **Update**: «Dashboard» > «Updates» > «Installieren»
- **Logs**: «Wartung» > «Protokolle»
