# Prozessdokument anlegen

## Voraussetzungen

- Berechtigungsgruppe: QMS-Ersteller oder QMS-Prozessverantwortliche
- Vorlage bereit (oder KI-Assistent nutzen)

## Schritte

1. Öffnen Sie die QMS-Dokumentenbibliothek
2. Klicken Sie auf «Neu» > «Prozessbeschreibung»
3. Füllen Sie die Pflichtfelder aus:
   - Prozessname
   - Prozessverantwortlicher
   - Gültigkeitsdauer
   - ISO-Kapitel (optional)
4. Laden Sie das Dokument hoch oder erstellen Sie es mit dem KI-Assistenten
5. Speichern Sie mit Status «Entwurf»

## KI-Assistent nutzen

Klicken Sie auf «Mit KI erstellen» und beschreiben Sie den Prozess in einfachen Worten.
