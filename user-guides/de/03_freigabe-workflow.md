# Freigabeprozess durchführen

## Als Ersteller: Freigabe einreichen

1. Öffnen Sie das Dokument im Status «Entwurf»
2. Klicken Sie auf «Freigabe beantragen»
3. Wählen Sie den Freigeber aus der Liste
4. Optional: Fügen Sie eine Notiz hinzu
5. Bestätigen Sie mit «Senden»

Der Freigeber erhält automatisch eine Benachrichtigung per Teams und E-Mail.

## Als Freigeber: Freigabe bearbeiten

Sie erhalten eine Adaptive Card in Teams mit den Dokumentdetails.

1. Prüfen Sie das Dokument (Link in der Karte)
2. Wählen Sie «Genehmigen» oder «Ablehnen»
3. Optional: Fügen Sie einen Kommentar hinzu
4. Bestätigen Sie

Bei Genehmigung wird das Dokument automatisch auf «Freigegeben» gesetzt.
