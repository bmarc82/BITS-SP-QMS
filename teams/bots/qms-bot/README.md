# QMS-Bot

## Befehle

| Befehl | Beschreibung |
|--------|-------------|
| `Hilfe` | Verfügbare Befehle anzeigen |
| `Aufgaben` | Meine offenen Reviews und Freigaben |
| `Suchen [Begriff]` | Prozess suchen |

## Proaktive Benachrichtigungen

Der Bot sendet Nachrichten bei:
- Neuer Freigabeanfrage
- Fälligem Review (60/30/7 Tage)
- Freigegebenem Dokument

## Technologie

- Bot Framework SDK
- Microsoft Teams Bot Channel
- Adaptive Cards v1.5
