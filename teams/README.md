# Teams Integration

Einbindung des QMS in den täglichen Arbeitskontext in Microsoft Teams.

## Komponenten

| Komponente | Beschreibung |
|-----------|-------------|
| App Manifest | Teams-App-Definition und Icons |
| process-map-tab | Prozesslandkarte als Teams-Tab |
| my-tasks-tab | Offene Aufgaben und Reviews |
| qms-bot | Proaktive Benachrichtigungen |
| process-search | Messaging Extension: Prozesssuche im Chat |
