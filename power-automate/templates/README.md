# Flow-Templates

Exportierte Power Automate Flow-Templates (.zip).

## Inhalt

- `QMS_ApprovalFlow_v1.0.zip` – Freigabe-Workflow
- `QMS_ReviewReminder_v1.0.zip` – Review-Erinnerung

## Import

1. Power Automate > Meine Flows > Importieren
2. ZIP-Datei hochladen
3. Verbindungen zu SharePoint und Teams konfigurieren
4. Importieren

> **Hinweis:** Die ZIP-Dateien werden nach dem ersten Deployment exportiert und hier abgelegt.
