# Power Automate Flows

Vollständige Automatisierung aller Genehmigungs-, Benachrichtigungs- und Eskalationsprozesse.

## Core Flows

| Flow | Beschreibung |
|------|-------------|
| approval-workflow | Freigabe-Workflow (Entwurf → Prüfung → Freigabe/Ablehnung) |
| review-reminder | Tägliche Prüfung, gestaffelte Erinnerungen 60/30/7 Tage |
| archive-old-versions | Automatische Archivierung von Vorgängerversionen |

## Modul-Flows

| Flow | Modul | Beschreibung |
|------|-------|-------------|
| risk-escalation | ISO 31000 | Eskalation bei kritischen Risikostufen |
| security-alert | ISO 27001 | Meldung bei neuen Sicherheitsdokumenten |
| ai-enrichment | KI | Metadaten-Anreicherung beim Upload |

## Import

Flow-Templates als .zip importieren:
1. Power Automate öffnen
2. «Importieren» > Template-ZIP hochladen
3. Verbindungen konfigurieren
