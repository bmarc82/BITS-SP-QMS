# QMS-Portal App

## Zweck
Vereinfachte Ansicht der Prozesslandkarte und geführter Workflow zur Dokumenterstellung.

## Screens

1. **Home** – Übersicht und Navigation
2. **Prozesslandkarte** – Gefilterte Prozessübersicht
3. **Dokument erstellen** – Geführter Erstellungsprozess
4. **Review-Übersicht** – Fällige Reviews (mobile-optimiert)
5. **Profil** – Persönliche Einstellungen

## Datenquellen

- SharePoint Liste: QMS-Prozesse
- SharePoint Bibliothek: QMS-Dokumente
- Office 365 Users
