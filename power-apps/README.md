# Power Apps

Mobile-freundliche Oberflächen für QMS-Aufgaben.

## Apps

| App | Zielgruppe | Beschreibung |
|-----|-----------|-------------|
| qms-portal | Alle Mitarbeitenden | QMS-Portal mit Prozesslandkarte |
| process-review-app | Process Owner | Mobile Review-App |
| audit-companion | Auditoren | Sichere Lesezugriff-App |

## Komponenten-Bibliothek

Wiederverwendbare UI-Elemente für alle Apps:
- `StatusBadge` – Farbiger Status-Indikator
- `ProcessCard` – Prozess-Übersichtskarte
- `ApprovalButtons` – Genehmigungs-/Ablehnungs-Buttons
