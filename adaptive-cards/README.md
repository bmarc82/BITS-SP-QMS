# Adaptive Cards

Einheitliche, interaktive Benachrichtigungs- und Aktionskarten für Teams, Power Automate und Outlook.

## Templates

| Template | Beschreibung |
|---------|-------------|
| approval-request.json | Freigabeanfrage mit Genehmigen/Ablehnen-Buttons |
| review-reminder.json | Review-Erinnerung mit Fristdatum und Direktlink |
| document-summary.json | Dokumentzusammenfassung |
| risk-alert.json | Risikowarnung (RM-Modul) |
| ai-suggestion.json | KI-Verbesserungsvorschlag mit Bewertungsoptionen |

## Vorschau

Öffnen Sie die HTML-Dateien in `preview/` für eine Entwicklungsvorschau.

## Schema-Version

Alle Karten verwenden Adaptive Cards Schema v1.5+
